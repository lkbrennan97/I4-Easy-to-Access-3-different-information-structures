# I4-Easy-to-Access-3-different-information-structures
I4 Easy to Access: Find 3 different information structures with different access technologies
https://canvas.uw.edu/courses/1883881/assignments/11160575

Find 3 different information structures with different access technologies (e.g. API, DB connection, file download and read locally). Write functions in a programming language of your choice to access each one and print out a sample of the information. In comments for each function, describe pros and cons of the access methodology. Upload code to Github and share URL, make sure any instructions needed to run are available in comments, preferably downloading and running the python is as simple as possible. 

Examples of different information structures:
    JSON
    XML
    Text
    HTML
    CSV
    PDF

Examples of different access technologies:

    API connection over HTTP for structured data from Site A
    Code library to read HTML over HTTP to view or download HTML
    SQL query database connection to export/save CSV
    Code library to connect to remote storage like AWS S3 or Azure Blob storage and download JSON/CSV/...
    Manual file download (from Kaggle for example) and read/load in Python

You can use any programming language.
